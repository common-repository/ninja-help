Need help with a WordPress plugin or a theme setting? Open this easy box and out pops live ninja to help!

== Description ==

You can easily ask for assistance with ease from your WordPress dashboard. Also signup to our newsletter from here with just a click. It is a live chat option so that you can ask any question you want about your WordPress site.

It hovers a button on the right hand side that will be accessible (and out of the way) on all WordPress admin pages. Easily accessible, easy to ask us a question.

== Installation ==

1. Upload '/ninja-help/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to "Ninja Help" on wp-admin RIGHT side menu.
